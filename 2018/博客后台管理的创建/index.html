hexo-admin博客的安装

选择hexo-admin

就目前而言，我所知道的发布Hexo的博客有如下几种：

1、原始方式，也就是在服务器上编写md文件，然后利用hexo g来生成；
2、利用github+hook来管理，小锅这里有一篇教程；
3、其他(hexo-admin or other plugins)；

安装

在Nodejs安装之后，安装了hexo，弄好了博客之后
npm install –save hexo-admin
hexo server -d

打开链接 http://localhost:4000/admin/
我打开之后直接可以访问后台管理，并没有博客主的登陆不了的问题，可能是因为本来就没配过账号密码，所以不知道怎么设置修改密码，用他的方法设置需要用shell脚本，那么windows客户端有点尴尬。

设置密码

按照提示把内容放进_config.yml
重启server之后 访问 http://localhost:4000/admin/就可以直接登陆了
